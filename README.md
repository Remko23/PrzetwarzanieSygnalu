# Cyfrowe przetwarzanie sygnału

# Zadanie 1

## Generacja sygnału i szumu

Celem ćwiczenia jest zapoznanie się z wybranymi własnościami podstawowych rodzajów sygnałów.

# Zadanie 2

## Próbkowanie i kwantyzacja

Celem ćwiczenia jest zapoznanie się z praktycznymi aspektami procesu konwersji analogowo-cyfrowej (A/C) i cyfrowo-analogowej (C/A) sygnałów. Zadanie polega na zaimplementowaniu procesu przetwarzania analogowo-cyfrowego z uwzględnieniem operacji próbkowania i kwantyzacji oraz konwersji odwrotnej, tj. cyfrowo-analogowej.

# Zadanie 3

## Splot, filtracja i korelacja sygnałów

Celem ćwiczenia jest zapoznanie się z wybranymi metodami cyfrowego przetwarzania sygnałów, obejmującymi operacje splotu, filtracji oraz korelacji sygnałów dyskretnych

# Zadanie 4

## Przekształcenie Fouriera, Walsha-Hadamarda, kosinusowe i falkowe, szybkie algorytmy.

Celem ćwiczenia jest zapoznanie się z operacjami transformacji sygnałów dyskretnych przy użyciu wybranych metod.
